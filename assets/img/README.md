# Imágenes de la web Dzongpa Europa

Esta carpeta contiene todas las imágenes de la web. Cada imagen sigue una nomenclatura estricta para facilitar su organización y referencia desde el código HTML.

## Convención de nombres

```
[código]-[descripción-corta].[extensión]
```

- **Todo en minúsculas**, sin espacios, sin acentos.
- **Guiones** para separar palabras.
- **Extensión** preferible: `.jpg` para fotos, `.png` para logo y gráficos con transparencia, `.webp` si la plataforma lo permite (más ligero).

## Notas sobre la infografía "Dzongpa Global"

La infografía promocional (art-01) es preciosa y reutilizable en redes sociales y como imagen de compartir (Open Graph). Atención: las cuatro tarjetas de la parte inferior muestran "Dzongpa Taipéi · Dzongpa Londres · Dzongpa Europa España · Dzongpa Europa España". Las dos últimas son idénticas — probablemente una de ellas debería ser "Dzongpa Australia" (ya que esa comunidad existe). Conviene corregirlo antes de usar la infografía como material oficial.

## Procesamiento técnico recomendado

Antes de subir cada imagen a la web:

1. **Renombrar** según este documento.
2. **Redimensionar** al tamaño máximo necesario (2400 px lado mayor para hero, 1200 px para contenido, 600 px para miniaturas).
3. **Comprimir** con calidad 80 % en JPG, o convertir a WebP.
4. **Generar versiones responsive** (2400, 1200, 800, 480 px) si la plataforma no lo hace automáticamente.
5. **Eliminar marcas de agua** donde existan (rin-03 tiene marca "buddhasweg.eu" en la esquina inferior izquierda — pedir versión sin marca al fotógrafo Hue Trieu).

## Cómo se referencian desde el HTML

Ejemplo de uso en una página:

```html
<img src="/assets/img/rinpoche/rin-02-rinpoche-altar-sonriente.jpg"
     alt="Su Eminencia el VI Gongkar Dorje Denpa Rinpoche en su residencia"
     width="1200" height="800" loading="lazy">
```

El nombre del archivo debe coincidir exactamente con el listado de arriba (sensible a mayúsculas y guiones).
